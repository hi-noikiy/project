如何充值聊币
聊币是什么？有什么用？
91NS的增值服务都是采用聊币结算，可以充值来获得聊币，用户也可以通过参加官方活动来获得聊币。1元人民币等于100聊币。充值购买的聊币永久有效。
如何充值？
91NS在导航栏、个人编辑信息弹窗、直播间礼物栏下方等处都提供了充值入口（{% if ns_userUid|default(false) %}传送门，戳我{% else %}传送门，戳我 {% endif %}）同时，充值方式，支持支付宝、微信、20多家银行储蓄卡及信用卡、手机充值卡的方式，方便安全。

充值未到账怎么办？
如果充值未到账，请您提供以下信息以便我们向相关部门查询。
1、91NS账号
2、充值交易号，可在个人中心-我的账单-充值记录里查询
网上银行不能使用怎么办？
部分银行只支持IE浏览器进行支付。当提示浏览器不兼容时，用IE浏览器打开91NS，在“{% if ns_userUid|default(false) %}充值记录{% else %}充值记录 {% endif %}”页面找到未支付的订单记录，点击去支付即可。
订单为什么会提示已过期？该怎么处理？
每个订单都有过期时间，一般为48小时。如果没有在有效期内支付该订单就会过期。过期后，需要重新充值。
为什么已完成付款，订单仍然显示未付款？
一般来说，网上支付是即时到账的。但有时因为网络原因可能会有延迟，稍后重新刷新页面即可。
